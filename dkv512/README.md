## DKV512 介绍

DVK512 是一块 Raspberry Pi Model B+的扩展板,它带有丰富的扩展接口,支持各类 外围模块的接入。

功能：

1. 4个LED
2. 4个按键
3. UART接口

...

## 安装依赖的库

```
sudo yum install libbcm2835 i2c-tools
```

> wiringPi必须要1.18以上版本才支持树莓派B+
> [安装方法](http://wiringpi.com/download-and-install/)
